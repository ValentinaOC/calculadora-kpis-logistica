# Calculadora-kpis-logistica
# Calculadora sencilla de Indicadores Logísticos 

## Descripción 

Esta es una aplicación interactiva desarrollada en Python con **Streamlit** diseñada para automatizar y calcular de manera sencilla los principales KPIs (Key Performance Indicators) de una operación logística, facilitando la visualizacion de errores y toma de desiciones basadas en los datos de una operacion real.

## problema
Los KPIS suelen hacerse manualmente en excel, consumiendo tiempo y aumentando el tiempo de errores.

## Solución
Aplicación que permite ingresar datos operativos y calcular automáticamente KPIs como:
* OTIF
* Pedidos a tiempo
* Nivel de servicio
* Exactitud de inventario
* Productividad de picking

## Tecnologías Utilizadas
* **Python** * **Streamlit** (para el despliegue de la interfaz web)

## Resultado
Reducción del tiempo calculado y estandarización de indicadores
